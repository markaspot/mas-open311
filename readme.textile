h2. REST-Plugin 

First get Kevin van Zonneveld's REST Plugin 
https://github.com/kvz/cakephp-rest-plugin

 
h2. Adding the plugin to your app_controller

Simply add the plugin as a another component 

pre. $component = array( ... ,'Open311.Open311');

h2. Adding custom REST-Routes

Open config/routes.php and paste the lines to your @routes.php@ located in your installation's config directory.
Service endpoint will be @/open311/@ as controller with custom actions in REST-Format
